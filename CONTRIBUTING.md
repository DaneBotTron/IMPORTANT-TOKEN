# Contributing

Thank you for contributing to **IMPORTANT TOKEN**.

This repository is intended to preserve and spread the IMPORTANT TOKEN message in a clear, stable, and reusable format.

## Ways to help

You can contribute by:

- improving clarity or formatting,
- fixing grammar or typos,
- suggesting a cleaner repository structure,
- adding documentation that helps others understand or share the message,
- improving the Python example or related tooling.

## Contribution guidelines

Please try to keep contributions:

- respectful,
- readable,
- focused,
- easy to copy and share,
- aligned with the core purpose of the repository.

## Before opening a pull request

Please make sure:

1. Your change fits the purpose of the project.
2. The wording is intentional and not confusing.
3. The repository still works as a simple public-facing message project.
4. Any technical additions stay lightweight unless clearly justified.

## Suggested process

1. Fork the repository.
2. Create a branch for your change.
3. Make your edits.
4. Commit with a clear message.
5. Open a pull request describing what changed and why.

## Questions or large changes

If you want to make a major change to the structure, tone, or message, open an issue first so the direction can be discussed before a large rewrite.

Thank you for helping preserve and improve this project.
