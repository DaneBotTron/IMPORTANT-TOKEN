# Security Policy

## Supported versions

This is a very small repository with minimal Python code and no external dependencies at this time.

| Version | Supported |
|---------|-----------|
| Latest  | Yes |

## Reporting a vulnerability

If you discover a legitimate security issue in the repository, please do not post exploit details publicly right away.

Instead:

1. Document the issue clearly.
2. Describe how it can be reproduced.
3. Include the file or component affected.
4. Send the report through the repository maintainer's preferred contact method, if one is listed.

## What counts as a security issue here

Examples may include:

- dangerous code execution behavior,
- accidental exposure of secrets,
- unsafe dependency additions in future updates,
- malicious or misleading code being introduced through contributions.

## Response goal

Reported issues should be reviewed and addressed as reasonably and quickly as possible.
